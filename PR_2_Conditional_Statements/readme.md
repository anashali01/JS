## üöÄ How to Run

1. **Clone or Download** this repository.
2. Open the `PR_2_Conditional_Statements` folder.
3. Run any program using Node.js:
   ```sh
   node <filename>.js
   ```
   Replace `<filename>` with the desired JS file (e.g., `prog1.js`).
